QuickTime Movie Parser


The QuickTime Movie Parser filter splits Apple® QuickTime® data into audio and video streams. The input pin connects to a source filter such as the File Source (Async) filter or the File Source (URL) filter. The Parser uses the Video For Windows (VFW) decompressor to decompress QuickTime files. If the VFW compressor does not support the compression method, it cannot render the data. The filter creates one output pin for the video stream and one output pin for the audio stream.
