Country/Region Codes and Channel to Frequency Mappings


The following information provides country/region codes, analog video standards, and channel to frequency mappings that are in use by most countries/regions in the world. The IAMTVTuner interface uses this information to set and view analog broadcast or cable channels that will be viewed through a Microsoft® DirectShow™ TV Tuner filter.
