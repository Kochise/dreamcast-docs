Multi-file Parser


The Multi-File Parser filter parses a simple file format that allows multiple actual filenames to be specified as though they were one file. This assists you in combining video and text for closed captioning. Also see SAMI (CC) Parser and Lyric Parser.

The Multi-File Parser files have the format shown in the following example: 


   ;MULTI
   http://server/share/video.mpg
   http://server/share/captions.smi
